## JavaScriptオブジェクト
キーがクォーテーション無し & 文字列をダブルクォーテーションで囲む
```
{ 
  id: 0,
  title: "title",
  description:  "desc"
}
```
キーがクォーテーション無し & 文字列をシングルクォーテーションで囲む
```
{ 
  id: 0,
  title: 'title',
  description:  'desc'
}
```


## JSONオブジェクト
キーをダブルクオーテーションで囲み、文字列をダブルクォーテーションで囲むのがJSON。 JavaScriptのオブジェクトでもある。
```
{
  "id": 0,
  "title": "title",
  "description":  "desc"
}
```
