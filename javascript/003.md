追加系
```
.unshift()：先頭に追加

.push()：末尾に追加
```

削除系
```
// .shift：先頭要素をリストから削除し、指定した変数に代入
arr = ["apple", "banana", "orange"];
a = arr.shift()
console.log(a) // apple
console.log(arr) // ["banana", "orange"]

.pop()：末尾要素をリストから削除し、指定した変数に代入
arr = ["apple", "banana", "orange"];
a = arr.pop()
console.log(a) // orange
console.log(arr) // ["apple", "banana"]
```
