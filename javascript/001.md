String()と.toString()とtoFixed()の違い

```
var num = 111;
```
String(num); // "111"
num.toString(10); // "111"
num.toFixed() // "111"
```

var num = 111.222;
```
String(num); // "111.222"
num.toString(10); // "111.222"
num.toFixed() // "111"
```

var num = -111.222;
```
String(num); // "-111.222"
num.toString(10); // "-111.222"
num.toFixed() // "-111"
